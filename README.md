<h2>File Integrity Monitoring w/PowerShell Scripts</h2>
<br />

<h2>Purpose</h2>
The main objective of this project is to emphasize the importance of monitoring unauthorized changes to a designated directory path. It tracks the hashes and paths of all files to identify any modifications, missing files, or newly created files, and notifies the user as needed.
